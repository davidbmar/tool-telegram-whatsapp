Setup Guide — Connect to Telegram

▼

Step 1: Create a Telegram Bot

  1. Open Telegram on your phone or desktop
  2. Search for @BotFather and start a chat with it
  3. Send the command: /newbot
  4. BotFather will ask for a name — type anything (e.g. My Sprint Bot)
  5. Then it asks for a username — must end in bot (e.g. mysprintbot)
  6. BotFather replies with your bot token — looks like 123456789:ABCdefGHI...
  7. Copy the token and paste it in the "Telegram Bot Token" field below

Step 2: Disable Privacy Mode (important!)

  1. Still in the BotFather chat, send: /mybots
  2. Select your bot from the list
  3. Tap Bot Settings
  4. Tap Group Privacy
  5. Tap Turn off — this lets the bot see messages in groups (needed to get the group ID)

Step 3: Create a Group and Add the Bot

  1. In Telegram, create a new group (tap the pencil/compose icon → New Group)
  2. Name it after your project (e.g. grassyknoll-dev)
  3. Add your bot as a member — search for the bot username you created (e.g. mysprintbot)
  4. Send a message in the group — type anything like "hello" (this is required for the next step)

Step 4: Get Your Group ID

After sending a message in the group, run this in your terminal:
curl https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates | python3 -m json.tool
Replace <YOUR_TOKEN> with the actual token from Step 1.
In the output, look for:
"chat": { "id": -100XXXXXXXXXX, "title": "grassyknoll-dev" }
The number starting with -100 is your group ID. Copy it.
Result is empty? Make sure you: (1) added the bot to the group, (2) disabled privacy mode in Step 2, (3) sent a message in the group AFTER adding the bot. Then try the curl command again.

Step 5: Configure and Test

  1. Scroll down to the Projects section below
  2. Change the Transport dropdown from "console" to telegram
  3. Paste your Group ID (the -100... number) into the Group ID field
  4. Click Save Configuration
  5. Click Send Test Message — you should see the message appear in your Telegram group!
